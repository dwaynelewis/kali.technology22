Getting Started

Mi. is a HTML/CSS template based on Bootstrap 3 framework. It allows to built impressive, better & faster, mobile friendly websites.

HTML Structure

This is the base of each HTML file:

<!-- Header -->
<header id="header">
    ...
</header>
<!-- Header / End -->


<!-- Section -->
  <section>
    ...
  </section>
<!-- Section / End -->

<!-- div -->
  <div>
    ...
  </div>
<!-- div / End -->

<!-- Footer -->
<footer id="footer">
    ...
</footer>
<!-- Footer / End -->
